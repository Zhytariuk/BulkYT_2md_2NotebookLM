# 🧪 Research Track: Битва з YouTube за субтитри

Цей файл описує шлях наших експериментів і гіпотез для тих, хто захоче продовжити роботу над проектом.

## 🔴 Issue 1: Блокування IP (Error 429) через youtube-transcript-api
- **Симптом**: Бібліотека видає `HTTP Error 429` навіть при першому запиті.
- **Гіпотеза**: YouTube бачить "голий" Python-запит без браузерних заголовків.
- **Експеримент**: Додано Cookies (MozillaCookieJar).
- **Результат**: Частковий успіх. Деякі відео пройшли, але масове блокування залишилось.

## 🔴 Issue 2: n-challenge (JavaScript захист)
- **Симптом**: `yt-dlp` видає попередження про відсутність JS-двигуна для розв'язання завдань.
- **Гіпотеза**: YouTube вимагає виконання JS для генерації посилання на субтитри.
- **Експеримент 1**: Встановлення Deno.
- **Експеримент 2**: Встановлення FFmpeg.
- **Результат**: `yt-dlp` у режимі API все одно пропускає завантаження компонентів (`skipped`), що призводить до відмови.

## 🔴 Issue 3: PO-Token (Proof of Origin)
- **Симптом**: Помилки про необхідність `GVS PO Token`.
- **Гіпотеза**: YouTube вимагає підпису від клієнтів `web`, `ios`, `android`.
- **Експеримент**: Переключення на `web_embedded` та `mweb` клієнти (План «E»).
- **Результат**: Блокування на рівні клієнта зберігається.

## 🚀 План «D/F»: Пряме завантаження (Manual Fetch) — ПОТОЧНИЙ СТАН
- **Гіпотеза**: Якщо ми самі витягнемо URL субтитрів і скачаємо його через `requests/urllib` з куками, ми обійдемо внутрішню логіку плеєра.
- **Реалізація**: 
    1. `yt-dlp` тільки для метаданих (це працює).
    2. Витягування `json3` або `vtt` посилання.
    3. Завантаження через `requests` з User-Agent та Cookies.
- **Результат**: Жорстке блокування IP (Код 429). YouTube бачить масовість запитів.

## 🏁 Резюме та рекомендації
На даний момент ми реалізували **максимально можливий рівень автоматизації**. Подальше просування вимагає або використання **проксі-серверів**, або **ручного отримання PO-Token** із браузера (що складно для звичайного користувача).

**Найкраща порада зараз: Робити паузи.** YouTube знімає 429-блок автоматично через 1-2 години.
